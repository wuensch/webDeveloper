HTML Classes

O atributo class do HTML serve para definir estilos para elementos com o mesmo nome de classe.

Assim, todos os elementos HTML com o mesmo atributo de classe, recebem o mesmo estilo.

O exemplo abaixo, 3 divs possuem o mesmo estilo.

Berlin
Tokyo
Cincinatti

Para associar a classe ao estilo, utiliza-se CSS e coloca-se um ponto (.) antes do nome do estilo.
